Agent 系统详细设计

Orchestrator 路由 + 两个专职子 Agent + 工具调用链

👤 用户消息
"帮我搜索 React 19 新特性并整理成笔记"
↓
🎯 Orchestrator Agent
1. 解析用户意图（搜索？管理？两者？）
2. 决定调用哪个子 Agent 及顺序
3. 汇总子 Agent 结果，流式返回给用户
模型选择器
● Qwen-Max (默认)
○ GPT-4o
○ Claude Sonnet
── 搜索意图 ──▶
🔍 搜索 Agent
工具调用链：
tool: tavily_search(query)
→ 返回网页摘要列表
↓
tool: summarize(results)
→ AI 提炼关键内容
↓
tool: save_article(md)
→ 写入 /content/
── 管理意图 ──▶
📂 管理 Agent
工具调用链：
tool: list_articles(filter)
→ 读取 /content/ 文件列表
↓
tool: categorize / tag
→ 更新文章 frontmatter
↓
tool: generate_summary
→ 生成/更新文章摘要
↓ 合并结果
⚡ 流式响应返回前端
Vercel AI SDK · StreamingTextResponse · useChat() hook
